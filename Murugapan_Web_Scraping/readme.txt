#Web Page Scraper

* A python script to parse the given web pages (mentioned in input text file - urls.txt) and extract the key vulnerablity metrices into JSON format.

* Libraries Imported - Beautiful Soup, Requests, lxml, Datetime

steps to run - 

1. using pip install downlaad and install beautiful soup, requests, lxml
2. place the input text file with urls in different lines in the working directory
3. run parser.py from the command line after navigating to the working directory
4. the output json is stored in text file(one each for each url) in the working directory, and also displayed in the cmd
